# EcoSphere – ESG Management Platform

This is an Odoo 18 module skeleton for the ESG (Environmental, Social, and Governance) Management Platform.

## Module Structure

- `models/`: ESG database schemas and logic skeletons
- `views/`: XML action, tree, form, and search views
- `security/`: XML user groups and access control lists (ACL)
- `controllers/`: HTTP controller routing definitions
- `data/`: Custom data and XML records
- `demo/`: Demonstration records
- `reports/`: PDF report action definitions
- `wizard/`: Transient wizard model skeletons
- `static/`: Frontend JS, CSS, and styling resources
- `tests/`: Automated unit and integration test definitions
- `i18n/`: Internationalization / translation files
